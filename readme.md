## About
Playback Info is a DisplayDuck widget that mirrors the built-in media display as a local pack. It shows the current title, artist, album, artwork, live state, and playback progress using the `@displayduck/media` package.

<p align="right">(<a href="#readme-top">back to top</a>)</p>


## Getting Started
No extra setup is required. Add the widget to a view and configure its display options to match your layout.


<p align="right">(<a href="#readme-top">back to top</a>)</p>

---

## Configurable options
| Setting | Type | Configurable Values | Default Value
|---|---|---|---|
| Auto-hide | `boolean` | `true`<br />`false` | `false`
| Orientation | `select` | `auto`<br />`vertical`<br />`horizontal` | `auto`
| Align Right | `boolean` | `true`<br />`false` | `false`
| Scroll Text | `boolean` | `true`<br />`false` | `false`
| Hide Cover | `boolean` | `true`<br />`false` | `false`
